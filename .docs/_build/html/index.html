Welcome to the OPNsense autodocs. Created using:

  • a parser that fetches the argParse arguments using arguments

  • bash and php sccripts where created with help of codeium - its just to much code for me - I checked most of the parameters needed if there are any

    • never the less theres still a room for missinpretation of missed constructor arguments for the scripts that where scanned manually

    • unfortunately there are not a lot of classes involed and the way the scripts parse the arguments differ quite often

  • the goal of this project was to create an overview about the scripts, the required parameters and flags
